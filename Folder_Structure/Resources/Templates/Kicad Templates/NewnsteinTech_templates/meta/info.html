This project template forms the basis of an expansion board (HAT) This includes the following models:

  • Pi 1 Model A+/B+
  • Pi 2 Model B
  • Pi 3 Model A+/B/B+
  • Pi 4 Model B

This project includes a PCB edge set. An alternate edge is included on the User.Drawings layer for the use with an SMT connector. Cutouts have also been defined for the camera and the display cables.
